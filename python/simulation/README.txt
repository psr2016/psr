
Elenco dei file presenti nella cartella
=======================================

* arm_control.py: controllo PID di un braccio robotico

* arm_response.py: simulazione di braccio robotico

* controllers.py: libreria con i vari tipi di controllori

* inputs.py: libreria con i vari tipi di segnali di ingresso

* proportional_controller.py: simulazione di controllo proporzionale

* proportional_controller_with_noise.py: simulazione di controllo
  proporzionale con "rumore"

* robot_response.py: simulazione di massa su piano + attrito aria

* speed_controller_with_saturation.py: simulazione di controllo PI in
  velocita' con saturazione

* speed_pi_controller.py: simulazione di controllore PI in velocita'

* step_controller.py: simulazione di controllore di tipo "step"

* systems.py: libreria di classi che implementano i modelli dei sistemi
  simulati
